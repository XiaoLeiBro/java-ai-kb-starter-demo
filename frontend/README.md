# 前端（占位）

当前项目按 **后端优先** 原则推进，前端暂为占位。

## 计划

| 阶段 | 前端形态 |
|---|---|
| v0.1（当前） | 无前端，用 `curl` 或 Postman 调接口 |
| v0.2 | 单文件 HTML + 简单 JS（Demo 录屏用） |
| v0.3 | Vue 3 + Element Plus 最简管理页 |
| 商业版 | Vue 3 + Pinia + Arco Design 完整管理后台 |

## 为什么先不做前端

- 验证 MVP 的核心是**后端链路跑通**，前端不是瓶颈
- 商业版用户更关心"后端源码 + 架构 + 二开能力"，前端差异化价值低
- 避免精力分散，符合文档 "9. 90 天落地计划" 原则
