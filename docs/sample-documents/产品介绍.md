# Java 企业 AI 助手产品介绍

## 产品定位

Java 企业 AI 助手是一套面向私有化部署、RAG 知识库、AI 客服和老系统二开的工程模板。

它适合帮助企业把内部文档、制度、产品资料、客服话术和培训资料接入 AI 问答。

## 核心能力

系统支持用户登录、知识库管理、文档上传、文本切分、向量化、向量检索、AI 问答、引用片段、对话历史和基础调用记录。

当前公开 Demo 支持 Markdown、TXT 和文本型 PDF。商业交付可以扩展扫描件 PDF OCR、Word、Excel 和 PPT 解析。

## 技术架构

后端基于 Java 和 Spring Boot。

数据库使用 PostgreSQL，并通过 pgvector 存储和检索向量。

前端使用 Vue 和 Element Plus。

系统可以通过 Docker Compose 启动后端、前端、PostgreSQL 和 Redis。

## 适用场景

适合企业制度问答、客服售后知识库、产品资料问答、培训机构课程问答、软件系统帮助中心和 Java 老系统 AI 改造。

不建议把当前 Demo 直接当成大型集团级生产系统使用。正式交付需要补齐权限、审计、安全、备份、监控和部署文档。

## 部署方式

系统支持本地演示部署和单机 Docker Compose 部署。

如果客户需要资料不出内网，可以配置本地模型或企业私有模型服务。

如果客户已有 Java 系统，可以按项目需求对接现有用户体系、文档中心、客服系统或业务后台。
