# 新员工入职指南

## 入职前准备

新员工入职前需要准备身份证、银行卡、学历证明、离职证明和一寸电子证件照。

如果岗位需要背景调查或资格证书，人事会在入职前单独通知。

新员工需要在入职前确认劳动合同信息、紧急联系人和社保公积金缴纳城市。

## 入职当天流程

新员工入职当天需要先到人事部门办理资料核验。

人事完成资料核验后，会协助员工签署劳动合同、保密协议和员工手册确认书。

行政部门会发放工牌、电脑、办公用品和门禁权限。

直属主管负责介绍团队成员、岗位职责、工作目标和试用期安排。

## 账号与权限

入职当天会开通企业邮箱、办公系统、代码仓库、项目管理系统和即时通讯账号。

涉及生产系统、财务系统、客户数据系统的权限，需要由直属主管发起申请，并经过部门负责人审批。

员工不得将账号借给他人使用，不得私自导出客户数据或公司内部资料。

## 试用期安排

试用期通常为 3 个月，具体以劳动合同为准。

试用期内，直属主管会安排入职培训、业务学习和阶段性任务。

试用期结束前，人事会组织转正评估。评估内容包括工作产出、协作表现、制度遵守和岗位匹配度。

## 常见问题

如果新员工发现账号无法登录，应先联系行政或 IT 支持。

如果入职资料暂时无法提供，应提前联系人事说明原因。

如果对薪资、社保、公积金或合同条款有疑问，应联系人事部门确认。
