# 商业版 Roadmap

## 定位

本项目不做通用知识库 SaaS，也不对标 Dify / FastGPT / MaxKB 做大而全平台。

商业化方向是：

```text
Java 企业 AI 助手工程模板：面向私有化部署、RAG 知识库、AI 客服和老系统二开的可交付 Demo。
```

Roadmap 的目标是服务接单、私有化部署和企业二开交付，而不是堆功能或追求开源 Star。

## 阶段一：可演示版

### 目标

让潜在客户和技术负责人看到完整闭环：

```text
登录 → 创建知识库 → 上传文档 → 向量化 → AI 问答 → 引用片段 → 对话历史 → 调用记录
```

### 目标客户

- 想快速看效果的小团队。
- 想验证 AI 知识库价值的老板或业务负责人。
- 想学习 Java AI 工程化的程序员。
- 想接单前准备演示模板的外包团队。

| Spring Boot 后端 | 已完成 |
| Vue + Element Plus 基础前端 | 已完成 |
| 用户注册 / 登录 / JWT | 已完成 |

| Markdown / TXT / 文本型 PDF 上传 | 已完成 |

| PostgreSQL + pgvector 检索 | 已完成 |
| RAG 问答与引用片段 | 已完成 |

| 客户介绍文档 | 已完成 |
| 演示脚本 | 已完成 |
| Demo 样本文档 | 已完成 |

### 交付标准

- 本地或服务器能启动完整演示环境。
- 能用样本文档完成 3 分钟演示。
- 能说明当前 Demo 与商业交付能力边界。
- 能给客户提出体验版试点方案。

## 阶段二：可交付版

### 目标

从 Demo 变成可以交给小客户试点的项目版本。

### 目标客户

- 电商客服团队。
- 培训机构。
- 小型 SaaS 公司。
- 咨询公司。
- 有内部制度、SOP、产品资料的小微企业。

### 计划能力

| 能力 | 说明 |
|---|---|
| 扫描件 PDF OCR | 支持图片型资料的文字识别 |
| Word / Excel 基础解析 | 根据客户资料类型选择性实现 |
| 文档删除与向量清理 | 删除文档后同步清理检索数据 |
| 文档重复上传处理 | 避免重复入库和重复召回 |
| 多用户基础权限 | 至少支持用户、管理员两类角色 |
| Token 用量统计 | 按用户、知识库、日期统计基础用量 |
| 调用日志增强 | 支持失败原因、耗时、模型、问题追踪 |
| 简单管理后台 | 查看用户、知识库、文档、调用记录 |
| 私有化部署文档 | 明确服务器、端口、环境变量、备份方式 |
| 交付检查清单 | 启动、登录、上传、问答、日志、备份验收 |

### 交付标准

- 客户能使用自己的少量真实资料试点。
- 业务负责人能判断回答质量是否可接受。
- 技术负责人能理解部署方式和数据流向。
- 能形成 5000～30000 元项目报价。

## 阶段三：企业增强版

### 目标

面向更正式的企业私有化部署和老系统 AI 改造。

### 目标客户

- 有现有 Java 系统的企业。
- 需要企业微信 / 飞书 / 钉钉入口的团队。
- 需要内网部署和审计的客户。
- 需要把 AI 能力接入业务流程的客户。

### 计划能力

| 能力 | 说明 |
|---|---|
| 完整 RBAC | 支持角色、资源、操作权限控制 |
| 组织架构 | 支持部门、成员、知识库归属 |
| 多租户 | 适合多客户或多组织隔离 |
| 多模型供应商 | DeepSeek、通义、硅基流动、OpenAI Compatible、Ollama |
| 本地模型部署方案 | 支持资料不出内网的客户 |
| 向量化任务监控 | 查看处理状态、失败原因、重试 |
| 限流 / 熔断 / 重试 | 提升模型调用稳定性 |
| 成本报表 | 按知识库、用户、模型、时间维度统计 |
| 审计日志 | 记录关键操作与问答行为 |
| 企业微信 / 飞书 / 钉钉接入 | 提供员工使用入口 |
| 客户现有系统集成 | 接入客服、文档中心、业务后台或网关 |

### 交付标准

- 需求边界、验收标准、部署环境在合同或确认单中明确。
- 具备部署文档、回滚方案、运维说明和培训材料。
- 能按里程碑交付，避免无边界定制。

## 暂不优先做的内容

以下能力可以进入长期规划，但不是当前第一阶段重点：

- 通用 SaaS 多租户平台。
- 在线支付和自动订阅。
- 大型低代码工作流。
- 全量 Agent 平台。
- 复杂 BI 报表。
- K8s 生产级编排。
- 多行业通用模板市场。

当前第一目标是拿到愿意看演示、愿意试用、愿意付费做体验版的客户。

## 推荐报价锚点

| 套餐 | 价格建议 | 目标 |
|---|---:|---|
| 体验版 | 5000～8000 元 | 3～5 天部署一套试点 Demo，验证真实需求 |
| 标准版 | 15000～30000 元 | 补齐常见文档解析、基础权限、日志和部署文档 |
| 私有化增强版 | 50000 元起 | 接入客户环境、模型、权限体系和业务系统 |

报价应根据资料格式、文档数量、部署环境、是否内网模型、是否对接现有系统调整。
